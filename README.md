# Tuning Digital

A static, SEO-optimised content site for **tuning websites, tools, and workflows**, built with Next.js and designed for AdSense monetisation and low maintenance.

## Features

- Next.js 14 static export
- Markdown-based blog
- Google AdSense (auto ads + blocks)
- Google Analytics 4
- Sitemap and robots.txt
- GitHub Actions deployment to GitHub Pages
- Claude-powered content generation via API + CLI

## Getting started

```bash
npm install
npm run dev
